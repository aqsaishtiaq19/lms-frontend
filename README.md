LMS Frontend
A modern Learning Management System frontend
built with React.js.

Features
- User Registration and Login
- JWT Authentication
- Course browsing and filtering
- Trending, Popular and New courses
- Add/Remove courses from Wishlist
- Student Dashboard
- Responsive design
- Password strength checker
- Show/Hide password toggle

Tech Stack
- React.js
- Styled Components
- React Bootstrap
- Axios
- React Router DOM
- React Icons

Pages
- Homepage with course slider
- Sign Up page
- Sign In page
- Student Dashboard
- Course listing page
- Categories page
- Wishlist page

Installation
```bash
npm install
npm start
```
